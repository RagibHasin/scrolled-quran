# Scrolled Quran 📜📖

Alhamdulillah!

All praise to Allah, Almighty, the Most Gracious, the Most Merciful.

This piece of software would have not been possible without the ["DigitalKhatt New Madina"](https://digitalkhatt.org/about) font which is a work of Dr. Amine Anan.

This application is built using the Xilem/Masonry framework from [Linebender](https://linebender.org/) community, which itself stands on the work of various many Rust crates.

Finally, the putting together of all these things has been the work of one Muhammad Ragib Hasin.
